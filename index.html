O que é o Arduino?

Arduino é uma plataforma de eletrónica usada para criar projetos interativos. Através de uma placa, sensores, atuadores e programação, é possível construir sistemas que reagem ao ambiente.

Educação

Ajuda a aprender eletrónica, programação e pensamento lógico através de experiências práticas.

Automação

Pode controlar luzes, motores, alarmes, rega automática e outros sistemas simples.

Prototipagem

Permite testar ideias rapidamente antes de criar um produto final.
